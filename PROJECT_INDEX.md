## 🎯 Breakpoint Reference

```
Mobile          320px - 640px    (base styles)
Small           640px            @media (min-width: 640px)
Tablet          768px            @media (min-width: 768px)
Desktop         1024px           @media (min-width: 1024px)
Large           1280px           @media (min-width: 1280px)
Extra Large     1536px           @media (min-width: 1536px)
```

**Layout Changes:**
```
Mobile:   1 column, stacked vertically
Tablet:   2 columns, side by side
Desktop:  3 columns, multi-column layouts
Large:    Adaptive auto-fit grids
```

```

Text:           #f5f5f5 (light)
Surface:        #252525 (slightly lighter dark)
Border:         #3a3a3a (dark gray)
```

- Primary on white: 20:1
- Secondary on white: 8.5:1
- Error on white: 7.2:1
